我们的翻译工作是从RC.4开始的，并且同样保持着同步更新。不过由于我们的译文要出版实体书，因此我们会在作者更新到Angular 2的正式版之后作为内容的截止点。不过可以放心，Angular 2历经两年半的开发之后，到发行正式版时API会完全定型，按照Angular 1的开发记录，即使有新的版本，也只会是功能扩展，API会保持向后兼容的。
